MCU Machine Learning

Below you'll see the results of machine learning for the Marvel Cinematic Universe.



MCU Heatmap

The first step was to create a heatmap to emphasize the correlation between all the variables using seaborn.

As you can see, the darker the color, the higher the positivie correlation. The more yellow the color is, the closer it is to negative correlation.

This image is condensed to emphasize the correlation by color, scroll down to see another seaborn correlation format.

Image



MCU Pair Plot

This is another generated image from seaborn called a pairplot. It does the same type of comparison as the heatmap, but instead of giving a color based on correlation, it shows what the graphs would look like for each comparison.

The bar chart graphs it created are comparing the same variables.

This was very helpful in planning which graphs to create to do machine learning on.
